# C. Расписание для взвешенных интервалов*

**Ограничение времени:** 5 секунд  
**Ограничение памяти:** 256 Мб  
**Ввод:** стандартный ввод или `input.txt`  
**Вывод:** стандартный вывод или `output.txt`  

---

Дан набор временных интервалов, у каждого из которых задан вес. Множество интервалов называется совместимым, если никакие два интервала в нем не пересекаются. Необходимо найти совместимое подмножество интервалов с максимальным весом.  
Под временными интервалами подразумеваются интервалы, не включающие свои концы.

## Формат ввода

В первой строке записано количество интервалов $N$ ($0≤N≤10^5$). В последующих $N$ строках на $i$-й строке записана информация об $i$-м интервале, т.е. три действительных числа: начало интервала $b_i$, конец интервала $e_i$ и его вес $w_i$. Известно, что для всех $i$ верно $b_i<e_i$, и $w_i>0$.

## Формат вывода

Выведите максимальный суммарный вес подмножества. Ответ будет зачтен, если он отличается от правильного не более чем на $10^{−4}$.

### Пример 1

**Ввод**
```
3
0 1 1
0.5 1.5 1.5
1 2 1
```
**Вывод**
```
2
```
